# Font Preview 

Минималистичное Android-приложение для удобного предпросмотра шрифтов в стиле **Material Design 3**.

## Особенности
* 🌍 **Онлайн каталог:** Доступ к 80+ популярным шрифтам из Google Fonts без необходимости скачивать их вручную.
* 📁 **Локальные файлы:** Загрузка и предпросмотр любых `.ttf` и `.otf` файлов с вашего устройства.
* 🎨 **Material Design 3:** Современный интерфейс с поддержкой динамических цветов (Dynamic Colors) и плавной анимацией.
* 📝 **Детальный предпросмотр:** Тестируйте шрифты на разных размерах (от 8sp до 72sp), проверяйте начертания и вводите свой собственный текст.
* 📱 **One UI Установка:** Инструкция и генератор пакета FlipFont для установки кастомного шрифта на устройства Samsung (через #mono_).
